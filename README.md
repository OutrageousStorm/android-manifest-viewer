# 🌐 Android Manifest Viewer

Browser-based tool to inspect `AndroidManifest.xml` from any APK or extracted manifest file.

**Live:** https://outrageousstorm.github.io/android-manifest-viewer/

## Features
- Drag & drop an APK or raw `AndroidManifest.xml`
- Highlights all `<uses-permission>` entries with danger level color-coding
- Lists all Activities, Services, Receivers, Providers
- Shows intent filters and exported components
- Works 100% client-side — nothing uploaded

## Use it
Just open `index.html` in any browser. No install needed.
